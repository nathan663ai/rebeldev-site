07 / FAQ

Straight answers.

The questions owners actually ask before they start.

01How long does it take?
Most builds land between four and twelve weeks, depending on scope. You get a realistic range at the discovery call, not an optimistic guess.
02Do I own the software?
Yes. On a one-off build you own the source code outright. On managed hosting you own your data and can take the software with you.
03What if the scope changes mid-build?
We expect it. Small changes are part of the process. Anything larger gets its own quick fixed price, so you always know what you are paying.
04Why not just hire a freelancer?
A freelancer disappears when the project ends. We build it, and can host it and stay on to fix and improve it if you want us to, so it keeps working long after launch.
05What happens if you disappear?
You keep everything. Your code and data are yours, documented and exportable. On a one-off build you can host it anywhere, with anyone.
